## カレンダーのアプリケーション
### 機能
- カレンダーを月ごとに表示
- 前後の月に切り替えること
- 日ごとに予定(Event)を登録すること
- 登録したEventの一覧表示
- 追加したEventの編集・削除

### 使用言語等
- Swift
- SwiftUI
- iOSシミュレーター 17.0
- Xcode-15.0.0
